# Product FAQ — SmartFlow SaaS

## Pricing
Starter plan: ₹999/month for up to 5 users.
Professional plan: ₹3,499/month for up to 25 users.
Enterprise plan: Custom pricing, contact sales@smartflow.io.

## Integrations
SmartFlow integrates with Slack, Jira, GitHub, Salesforce, and Zoho CRM.
API rate limit is 1000 requests/hour on Professional plan.
Webhooks are available on Professional and Enterprise plans only.

## Data & Security
All data is encrypted at rest using AES-256 and in transit using TLS 1.3.
We are SOC2 Type II certified and GDPR compliant.
Data is hosted in Mumbai (ap-south-1) AWS region.
Daily backups are retained for 30 days.

## Support
Starter plan: Email support, 48-hour response SLA.
Professional plan: Email + chat support, 4-hour response SLA.
Enterprise plan: Dedicated account manager, 1-hour SLA, phone support.
